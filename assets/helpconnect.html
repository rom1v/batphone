Serval Connect

There are two checkbox options.
These will, if checked, however, drain your battery while connected to a network, even when the screen is off.
This is because they will keep cycling through, attempting to make connections (as described below).
Without a network connection to at least another phone, the Serval Mesh software connot provide any useful function.

Enable ServicesThis, when checked, will turn everything Serval related on when you connect to any WiFi network.
This option also displays the current status of services running.
This, when checked, when enabled, a further feature shows the number of available phones on the network.
Auto CycleWhen there is no network connection, or no detected peers, (available phones), this will periodically cycle between Hot Spot & Adhoc modes to allow network connection between nearby phones.

Below these checkboxes is displayed the list of networks that you can attempt to connect to. Simply clicking on one will attempt the connection.
